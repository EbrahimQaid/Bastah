Act as a senior product designer, UX expert, and full-stack architect.

Your task is to design and generate a premium, high-converting, mobile-first multi-vendor e-commerce web application tailored for small sellers who primarily sell through WhatsApp.

The platform should feel as polished and high-end as top global e-commerce platforms, but significantly simpler, faster, and optimized for non-technical users.

--------------------------------------------------

🎯 CORE GOAL:
Create a seamless, elegant shopping experience for customers while keeping WhatsApp as a hidden backend communication tool for order delivery.

--------------------------------------------------

🎨 DESIGN SYSTEM:

Use a modern, clean, minimal, and premium visual style.

Color palette:
- Primary: Deep red (#C1121F)
- Background: White (#FFFFFF)
- Surface: Light gray (#F3F4F6)
- Text primary: Dark gray (#1F2937)
- Text secondary: Gray (#6B7280)

Design principles:
- Minimal and uncluttered
- Generous whitespace
- Smooth visual hierarchy
- Rounded corners (medium to large radius)
- Soft shadows (subtle elevation)
- High-end typography (modern sans-serif)

--------------------------------------------------

✨ UX & INTERACTIONS:

- Mobile-first design (priority)
- Fast and responsive UI
- Smooth transitions and micro-interactions
- Subtle hover and tap animations
- Instant feedback on actions (add to cart, clicks)

--------------------------------------------------

🛍️ CUSTOMER EXPERIENCE:

1. Home Page:
- Clean hero section
- Featured products
- Categories (clear and scrollable)
- Promotional banners (minimal)

2. Search:
- Smart live search (instant suggestions)
- Typo-tolerant
- Search by name, category, or keywords

3. Filters:
- Price range slider
- Category selection
- Size options
- Color selection (visual swatches)

4. Product Listing:
- Grid layout
- High-quality product cards
- Image, name, price
- Hover/tap animation

5. Product Page:
- Large image gallery (swipe enabled)
- Zoom interaction
- Product title and price
- Variant selectors:
  - Sizes (buttons)
  - Colors (color swatches)
- Stock indicator ("Only few left")
- Ratings and reviews
- Strong CTA button (Add to Cart)

6. Cart:
- Smooth add-to-cart animation
- Editable cart (quantity, remove)
- Clear pricing breakdown
- Sticky checkout button

7. Checkout:
- Minimal friction form:
  - Name
  - Phone
  - Address
  - Notes
- Clean layout
- Premium “Place Order” button
- DO NOT mention WhatsApp in UI
- Seamless completion flow

--------------------------------------------------

🔄 BACKEND LOGIC (IMPORTANT):

- Automatically generate a structured WhatsApp message after checkout
- Include:
  - Customer info
  - Product names
  - Variants (size, color)
  - Quantities
  - Total price

- Ensure the process feels native and invisible to the user

--------------------------------------------------

🎛️ SELLER DASHBOARD:

Design an extremely simple, fast, and mobile-friendly dashboard.

Features:

1. Products:
- Add/edit/delete products quickly
- Upload multiple images
- Add variants easily (size, color)

2. Categories:
- Simple category management

3. Orders:
- List of orders
- Status:
  - New
  - Contacted
  - Completed
- Button: Open in WhatsApp

4. Store Settings:
- Store name
- Cover image
- Theme colors

5. Utilities:
- Copy store link
- Quick actions

--------------------------------------------------

💎 PREMIUM EXPERIENCE DETAILS:

- Micro-interactions everywhere (buttons, cards, transitions)
- Smooth animations (not heavy)
- Elegant spacing and alignment
- Clean, distraction-free UI
- Trust-building elements:
  - Ratings
  - Stock indicators
  - Clean layout

--------------------------------------------------

⚡ PERFORMANCE:

- Fast loading
- Lightweight UI
- Optimized for low الإنترنت speeds
- Mobile performance is critical

--------------------------------------------------

🎯 FINAL OBJECTIVE:

Build a product that:
- Feels premium and modern
- Is extremely easy for sellers to use
- Makes customers trust the store instantly
- Combines the power of e-commerce with the simplicity of WhatsApp selling

--------------------------------------------------

Deliver:
- Complete UI design
